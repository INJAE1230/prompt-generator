# Claude Code 프로젝트 지침

## 모델 전환 제안

작업 요청을 받으면 복잡도를 판단해 적합한 모델을 제안한다.

**단순 작업 → `/model sonnet` 제안:**
- 필드 추가, 타입 수정, 상수 변경 등 단순 편집
- 기존 패턴을 반복 적용하는 수정
- 단일 파일 내 간단한 쿼리나 함수 추가
- 오타 수정, 스타일 변경

**복잡한 작업 → `/model opus` 권유:**
- 설계·아키텍처 판단이 필요한 작업
- 여러 파일에 걸친 리팩토링
- 긴 맥락 유지가 필요한 작업 (복잡한 버그 추적, 대규모 기능 구현)
- 트레이드오프 분석, 성능 최적화 전략 수립

현재 모델이 적합하면 별도 언급 없이 바로 진행한다.
